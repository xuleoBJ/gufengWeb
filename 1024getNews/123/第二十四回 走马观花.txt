    宝玉为姑娘斟了酒，道：“这酒叫‘梨花白’，清冽甘甜，且一点不辣，最适合女孩子吃。”见采容客气，便又笑道：“我点的这几道菜，皆是都中有名的小食，这里做得还算j致，姐姐且尝尝。”<br/>
<br/>
    淩采容肚子饿得再挨不过，拎箸朵颐，尝了几样，皆是十分可口美味，吃得心畅口滑，竟嫌凤姐的衣裳拖曳累人，忽卷起袖子，露出一载滑雪雪的耦臂，哪里还有方才的淑女模样。<br/>
<br/>
    宝玉瞧得目瞪口呆，只觉面前女孩，比起家里的姐妹们另有一种风情味道，心里那老毛病又犯，便有些痴傻起来。<br/>
<br/>
    淩采容吃得津津有味，那“梨花白”也一杯接一杯地畅饮，只觉入口十分容易，喝得俏脸微晕，无意间擡头，眼角瞥见对面公子的那副尊容，自已倒有点不好意思起来，道：“你怎么不吃呀？”<br/>
<br/>
    宝玉这才拾箸夹食，兴许是受了她的感染，吃起来竟觉比平日格外有胃口。<br/>
<br/>
    淩采容见两人没话的吃，不觉有点尴尬，黑漆漆的眼珠子一转，用箸指着一碟菜，寻了个话题：“这是什么呢？这样好吃。”<br/>
<br/>
    宝玉道：“这叫‘炒肝儿’，最先出自禁城前门外的‘会仙居’，原叫‘白水杂碎’，用切成段的猪肠、肝、心、肺，加调料用白汤煮就。出名后，都中四九城的饭馆酒楼都相继添了这道菜，市面上也出现了以‘炒肝儿’为说词的俏皮活，如骂人时便说‘你这人怎么跟炒肝儿似的，没心没肺。’讽刺互相残害的人与事，则说‘猪八戒吃炒肝，自残骨r。’”<br/>
<br/>
    淩采容听了这些典故，只觉十分有趣，笑吟吟自语道：“炒肝儿……没心没肺。”又咬着箸尖道：“这些都是常用材料，却做得这般好吃，难怪能够如此盛行。”<br/>
<br/>
    宝玉道：“材料是普通，但要做得好却不太容易哩！以前曾叫厨房的来问过，说是先将猪肠用碱、盐泡揉搓，用清水加醋洗净后再煮。开锅后改用文火肫，锅盖盖好使肠子熟透而不跑油。烂熟后，切成五分长的小段，俗称‘顶针段’，再将鲜猪肝洗净，用刀斜片成柳叶形的条。佐料是熬热的食油中放入大科，炸透后放入生蒜，蒜变黄时立即放入适量的黄酱，炒好置于罐中备用。此外还要熬些上好的口磨汤。原料、佐料备齐后，始制作炒肝儿。先将熟肠段放入沸汤，再放入蒜酱、葱花、芡同姜末和口磨汤，然后路生肝条放入锅中，以淀粉勾芡，最后撒上一层砸好的蒜泥即成。”<br/>
<br/>
    淩采容吐吐舌道：“这么多道工序，难为你记得住。”<br/>
<br/>
    宝玉笑道：“说起来繁复，但若是跟我家里做的许多菜肴比起来，却又算简单了，那些才叫做折腾人哩！”<br/>
<br/>
    淩采容听到这，忽道：“对了，你家里有什么大官？那么大的一座府第哩！”<br/>
<br/>
    宝玉简单扼要说道：“我祖上是荣国公，爷爷是京营节度使世袭一等神威将军，我爹是工部员外郎。”<br/>
<br/>
    淩采容吐吐舌头道：“听起来好像都是大官呀，你将来也会做大官是么？”宝玉皱皱眉头，却闭了口，再不愿聊此话题。<br/>
<br/>
    淩采容瞧瞧他，识趣的又指一道菜，问道：“这是又是什么？也很好吃哩！”<br/>
<br/>
    宝玉道：“这叫‘薰鱼儿’，就是用黄花鱼……”<br/>
<br/>
    突听旁边“砰”的一声巨响，皆把两人吓了一跳，转首瞧去，只见不远的桌子，一高大身影猛地立起来，怒气冲霄地喝道：“他***，这算哪门子鸟事！连那个小小的‘正心武馆’都有人被邀去东太师府赴宴，而我们名震华东五省的‘车马会’却一个没请，一个个在这里吃自已，真不知他们是怎么瞧人的！”</div>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<br/>
    贾母笑搂着宝玉，道：“等你猴子的孙儿也这样大的时候，都不知何年何月了。”<br/>
<br/>

<br/>

<br/>

<br/>

<br/>

<br/>

<div class="yd_text2">    ｛正心武馆人物｝<br/>
<br/>
    殷正龙——正心武馆馆主。原出自少林，法号“无心”，是“无”字辈中的佼佼者，在少林短短的十几年间，已习得少林正三十六房绝技中的六房，其中看似最平凡的一路“伏虎拳”更是给他修习得炉火纯青，曾被少林寺罗汉堂圣僧了空赞誉：“近千年来伏虎拳第二人”。<br/>
<br/>
    武功：少林六房<br/>
<br/>
    林慧嫱——殷正龙之妻，乃武林六大世家中九江林家的传人，识得许多武林典故。<br/>
<br/>
    武功：林家裙裏腿<br/>
<br/>
    殷琳——殷正龙夫妇之女。明豔端丽，脸上生著一对灵气四溢的大眼睛，是个令白玄、冷然与宝玉皆心仪的女孩子。<br/>
<br/>
    主线人物之一。<br/>
<br/>
    最近战绩：在去为龙盟主拜夀的途中遇见白莲剑妖，幸给冷然搭救，并“联手”诛之，哄动江湖。<br/>
<br/>
    邹远山——正心武馆大弟子。<br/>
<br/>
    武功：少林三十六房之一的伏魔金刚环<br/>
<br/>
    兵器：铁双环<br/>
<br/>
    白玄——正心武馆弟子。因机缘得入丁翊秘库，寻得“凤凰涅盘大法”的秘籍，又一举击杀江南五名采花大盗，获得《无极谱》、收魂散、碧玉戒、美人眸等数样异宝，後因贪念踏入丁翊秘库的十八层地狱图当中，惨遭十八铜偶群攻而殁。<br/>
<br/>
    古立、阿竹、大水牛、翁敏——皆为正心武馆弟子。<br/>
<br/>

    ｛其他人物｝<br/>
<br/>
    不死罗刹——天竺第一高手，八十年前踏足中原，向武林各门派的名家好手挑战，并在短短的半年内打遍江湖无敌手。後来又上少林，击败四大护法圣僧， 并向当时少林寺唯一修炼成“易筋经”的僧人无为挑战。<br/>
<br/>
    是役无人知晓胜负，此後无为闭关直至圆寂，而她亦从此在江湖上销声匿迹，下落不明。<br/>
<br/>
